JavaScript Introduction

What is JavaScript?
JavaScript is the programming language of the web. It allows you to update and change both HTML and CSS, manipulate and validate data, and program the behavior of web pages.

Why Study JavaScript?
JavaScript is one of the 3 core technologies every web developer must learn:

  • HTML – Defines the structure and content
  • CSS – Defines the layout and design
  • JavaScript – Defines the interactive behavior

JavaScript Can Change HTML Content:

document.getElementById("demo").innerHTML = "Hello JavaScript";

JavaScript Can Use Single or Double Quotes:

document.getElementById('demo').innerHTML = 'Hello JavaScript';

JavaScript Can Change HTML Attribute Values:
Example: Changing the src of an image tag. 

document.getElementById("myImage").src = "pic_bulbon.gif";

JavaScript Can Change HTML Styles (CSS):

document.getElementById("demo").style.fontSize = "35px";

JavaScript Can Hide HTML Elements:

document.getElementById("demo").style.display = "none";

JavaScript Can Show HTML Elements:

document.getElementById("demo").style.display = "block";

JS Where To

JavaScript code can be placed in an HTML document using the <script> tag. It can appear in the <head> or <body> section, or be included as an external file.

Inline Script Example

<script>
document.getElementById("demo").innerHTML = "My First JavaScript";
</script>

JavaScript in <head>

<head>
  <script>
    function myFunction() {
      document.getElementById("demo").innerHTML = "Paragraph changed.";
    }
  </script>
</head>
<body>
  <p id="demo">A Paragraph</p>
  <button onclick="myFunction()">Try it</button>
</body>

JavaScript in <body>

<body>
  <p id="demo">A Paragraph</p>
  <button onclick="myFunction()">Try it</button>
  
  <script>
    function myFunction() {
      document.getElementById("demo").innerHTML = "Paragraph changed.";
    }
  </script>
</body>

External JavaScript File

You can write JavaScript in a separate file with a .js extension and include it in your HTML file:

<script src="myScript.js"></script>

Example of external JS file content (myScript.js):

function myFunction() {
  document.getElementById("demo").innerHTML = "Paragraph changed.";
}

External Script References

  • <script src="myScript.js"></script> – Local file in the same folder
  • <script src="/js/myScript.js"></script> – File in /js/ directory
  • <script src="https://example.com/js/myScript.js"></script> – External file from full URL

Advantages of External JavaScript:

  • Separates HTML from logic
  • Improves code reusability
  • Allows browser caching for faster performance

JavaScript can display data in various ways:

  • Using innerHTML or innerText
  • Using document.write()
  • Using alert()

  • Using window.print() for printing

JS Statements

JavaScript statements are instructions that tell the browser what to do. They form the building blocks of JavaScript code.

Example

Semicolons

Semicolons ; are used to separate JavaScript statements.

Multiple statements can also be written on one line:

a = 5; b = 6; c = a + b;

White Space and Readability

JavaScript ignores extra white spaces. Use spaces to make code more readable:

let person = "Hege";
let person="Hege"; // Same as above

Line Breaks

For readability, break long lines after an operator:

document.getElementById("demo").innerHTML =
"Hello Dolly!";

Code Blocks

Use curly braces { } to group multiple statements together:

function myFunction() {
  document.getElementById("demo1").innerHTML = "Hello Dolly!";
  document.getElementById("demo2").innerHTML = "How are you?";
}

JS Variables

JavaScript variables are containers for storing data values. You can declare them using var, let, or const.

Declaring Variables

var x = 5;
let y = 10;
const z = 15;

var – declares a variable globally or locally to an entire function regardless of block scope.

let – declares a block-scoped variable, recommended over var for block safety.

const – declares a constant, block-scoped variable whose value cannot be changed.

Reassigning Values

let name = "John";
name = "Doe";  // This is valid

const country = "India";
// country = "USA";  // ❌ This will throw an error

Multiple Declarations

let a = 1, b = 2, c = 3;

Variable Naming Rules

  • Must begin with a letter, $, or _
  • Case sensitive (e.g., myVar and MyVar are different)
  • Cannot use reserved keywords (like let, class) as variable names

JS Let

let allows you to declare block-scoped variables. It is preferred over var in modern JavaScript.

Example:

let x = 10;
if (x === 10) {
  let y = 20;
  console.log(y); // 20
}
console.log(y); // ReferenceError: y is not defined

Why use let?

  • It respects block scope.
  • Cannot be redeclared in the same scope.
  • Prevents accidental variable overwrites.

JS Const

const is used to declare constants (read-only variables). You must assign a value during declaration.

Example:

const pi = 3.14159;
console.log(pi); // 3.14159

pi = 3.14; // TypeError: Assignment to constant variable.

Important:

  • const variables cannot be reassigned.
  • They are block scoped like let.
  • However, if the value is an object or array, its properties or items can be changed.

Example with Object:

const person = { name: "Alice" };
person.name = "Bob"; // Allowed
console.log(person.name); // "Bob"

JS Operators

Operators are used to perform operations on variables and values.

Types of JavaScript Operators:

  • Arithmetic Operators: +, -, *, /, %, ++, --
  • Assignment Operators: =, +=, -=, *=, /=, %=
  • Comparison Operators: ==, ===, !=, !==, >, <, >=, <=
  • Logical Operators: &&, ||, !
  • String Operators: +, +=
  • Type Operators: typeof, instanceof

Example:

let x = 10;
let y = 5;

console.log(x + y);  // 15 (Arithmetic)
console.log(x > y);   // true (Comparison)
console.log(x == 10 && y == 5); // true (Logical)

JS Arithmetic

JavaScript supports standard arithmetic operators for performing mathematical calculations.

Arithmetic Operators:

  • + : Addition
  • - : Subtraction
  • * : Multiplication
  • / : Division
  • % : Modulus (Remainder)
  • ++ : Increment
  • -- : Decrement

Example:

let a = 10;
let b = 4;

console.log(a + b); // 14
console.log(a - b); // 6
console.log(a * b); // 40
console.log(a / b); // 2.5
console.log(a % b); // 2

JS Assignment

Assignment operators are used to assign values to JavaScript variables.

Common Assignment Operators:

  • = : Assigns value
  • += : Adds and assigns
  • -= : Subtracts and assigns
  • *= : Multiplies and assigns
  • /= : Divides and assigns
  • %= : Modulus and assigns

Example:

let x = 10;
x += 5;  // x = x + 5 → x = 15
x -= 3;  // x = x - 3 → x = 12
x *= 2;  // x = x * 2 → x = 24
x /= 4;  // x = x / 4 → x = 6
x %= 4;  // x = x % 4 → x = 2

  • Number – represents integer and floating-point numbers
  • BigInt – for large integers

JS Functions

Functions in JavaScript are blocks of code designed to perform a specific task. They are executed when "called".

Declaring a Function

You can declare a function using the function keyword:

function greet() {
  console.log("Hello!");
}

Calling a Function

Once defined, you can call the function by its name followed by parentheses:

greet(); // Outputs: Hello!

Function Parameters

You can pass parameters to a function to work with dynamic values:

function greetUser(name) {
  console.log("Hello, " + name + "!");
}
greetUser("Alice"); // Outputs: Hello, Alice!

Return Statement

A function can return a value using the return keyword:

function add(a, b) {
  return a + b;
}
let result = add(5, 3); // result is 8

Arrow Functions (ES6)

Arrow functions provide a shorter syntax:

const multiply = (x, y) => x * y;
console.log(multiply(4, 2)); // Outputs: 8

JS Objects

Objects in JavaScript are used to store keyed collections of various data and more complex entities.

Creating an Object

You can create an object using curly braces {} and define key-value pairs:

const person = {
  firstName: "John",
  lastName: "Doe",
  age: 30,
  isStudent: false
};

Accessing Object Properties

Use dot notation or bracket notation to access values:

console.log(person.firstName);     // John
console.log(person["lastName"]);   // Doe

Modifying Object Properties

You can update values using assignment:

person.age = 31;

Adding New Properties

person.nationality = "American";

Object Methods

You can define functions inside objects, called methods:

const person = {
  firstName: "Alice",
  lastName: "Smith",
  fullName: function() {
    return this.firstName + " " + this.lastName;
  }
};

console.log(person.fullName()); // Alice Smith

The this Keyword

In an object method, this refers to the object itself.

Built-in Object Examples

  • Math: for mathematical operations
  • Date: for working with dates
  • Array: for list-like objects

JS Object Methods

Object methods are functions stored as object properties. They allow objects to perform actions using their own data.

Defining Methods in Objects

const person = {
  firstName: "John",
  lastName: "Doe",
  fullName: function() {
    return this.firstName + " " + this.lastName;
  }
};

console.log(person.fullName()); // John Doe

Shorter Method Syntax

You can define methods using ES6 shorthand syntax:

const person = {
  firstName: "Jane",
  lastName: "Smith",
  fullName() {
    return `${this.firstName} ${this.lastName}`;
  }
};

Using this Keyword

The this keyword refers to the current object context:

const car = {
  brand: "Toyota",
  model: "Corolla",
  getInfo() {
    return this.brand + " " + this.model;
  }
};

console.log(car.getInfo()); // Toyota Corolla

Adding Methods to Existing Objects

const user = {};
user.sayHello = function() {
  return "Hello!";
};

console.log(user.sayHello());

JavaScript objects can be displayed using various techniques like accessing individual properties or converting the object to a string.

To convert the whole object into a string:

JS Object Constructors

In JavaScript, object constructors are templates used to create multiple similar objects. A constructor function defines properties and methods for an object.

Example: Creating a Constructor Function


function Person(firstName, lastName, age) {
  this.firstName = firstName;
  this.lastName = lastName;
  this.age = age;
  this.fullName = function() {
    return this.firstName + " " + this.lastName;
  };
}

const person1 = new Person("John", "Doe", 30);
const person2 = new Person("Jane", "Smith", 25);

console.log(person1.fullName()); // Output: John Doe
console.log(person2.age);        // Output: 25

- Use the new keyword to create instances.
- Each object created has its own properties and methods.
- Constructor functions typically start with a capital letter by convention.

JS Strings

In JavaScript, strings are used to store and manipulate text. A string is any text inside quotes — single or double.

Creating Strings


let name1 = "John";

String Concatenation

You can combine strings using the + operator:


let firstName = "John";

String Length

The length property returns the length of a string:

Accessing Characters

You can access characters in a string using bracket notation or the charAt() method:

Use backticks ` ` to embed expressions inside strings:

JS String Methods

JavaScript provides many useful methods to work with strings. Below are some commonly used string methods.

toUpperCase() and toLowerCase()

Convert strings to upper or lower case:


let text = "JavaScript";
console.log(text.toUpperCase()); // "JAVASCRIPT"
console.log(text.toLowerCase()); // "javascript"

indexOf()

Returns the position of the first occurrence of a specified value:


let str = "Hello, world!";
console.log(str.indexOf("world")); // 7

slice()

Extracts a section of a string:


let str = "Hello, JavaScript!";
console.log(str.slice(7, 17)); // "JavaScript"

replace()

Replaces a specified value with another value:


let str = "I love cats!";
let newStr = str.replace("cats", "dogs");
console.log(newStr); // "I love dogs!"

trim()

Removes whitespace from both sides of a string:


let str = "   Hello World!   ";
console.log(str.trim()); // "Hello World!"

split()

Splits a string into an array:


let str = "apple,banana,orange";
let arr = str.split(",");
console.log(arr); // ["apple", "banana", "orange"]

JS String Search

JavaScript offers several methods to search within strings.

indexOf()

Finds the first occurrence of a specified value and returns its position:


let str = "Please locate 'locate' in this string.";
console.log(str.indexOf("locate")); // 7

lastIndexOf()

Returns the position of the last occurrence of a specified value:


console.log(str.lastIndexOf("locate")); // 15

search()

Searches for a match using a regular expression:


let result = str.search(/locate/);
console.log(result); // 7

includes()

Checks if a string contains a specified value (case-sensitive):


let found = str.includes("locate");
console.log(found); // true

startsWith()

Checks if a string starts with a specified value:


let greeting = "Hello world!";
console.log(greeting.startsWith("Hello")); // true

endsWith()

Checks if a string ends with a specified value:


console.log(greeting.endsWith("world!")); // true

JS String Templates

JavaScript Template Literals (Template Strings) allow you to embed expressions and variables inside strings using backticks (`).

Basic Syntax

Use backticks instead of single or double quotes:


let name = "John";
let greeting = `Hello, ${name}!`;
console.log(greeting); // Hello, John!

Multi-line Strings

You can create multi-line strings easily with template literals:


let text = `This is
a multi-line
string.`;
console.log(text);

Expression Substitution

You can embed any valid JavaScript expression inside ${...}:


let a = 5;

Function Calls

Functions can also be embedded:


function greet() {
  return "Welcome!";
}
let message = `${greet()} John.`;
console.log(message); // Welcome! John.

JS BigInt

JavaScript BigInt is a special numeric type that can represent integers with arbitrary precision. It is used when numbers exceed the safe limit of the Number type.

Creating a BigInt

Use n at the end of the integer or use BigInt():


// Using 'n' suffix
let big1 = 1234567890123456789012345678901234567890n;

// Using BigInt() constructor
let big2 = BigInt("1234567890123456789012345678901234567890");

console.log(big1);
console.log(big2);

Operations with BigInt

You can perform arithmetic with BigInt, but only with other BigInts:


let a = 9007199254740991n;
let b = 1n;
let sum = a + b;

console.log(sum); // 9007199254740992n

Important Notes

  • BigInt cannot be mixed with regular Number types.
  • Use .toString() or String() to convert BigInt to string.

let x = 10n;
// console.log(x + 5); ❌ TypeError
console.log(x + 5n); // ✅ 15n

console.log(isNaN("abc")); // true

Common Number Properties

Returns the largest possible number in JavaScript:


console.log(Number.MAX_VALUE);

JS Arrays

JavaScript arrays are used to store multiple values in a single variable. Arrays are dynamic and can hold items of any type.

Creating an Array


// Using square brackets
let fruits = ["Apple", "Banana", "Mango"];

// Using the Array constructor
let numbers = new Array(1, 2, 3, 4);

Accessing Array Elements

Array elements are accessed using index numbers (starting from 0):


console.log(fruits[0]); // Output: Apple
console.log(fruits[2]); // Output: Mango

Modifying Array Elements


fruits[1] = "Orange";
console.log(fruits); // Output: ["Apple", "Orange", "Mango"]

Array Properties

  • length: Returns the number of elements in the array

console.log(fruits.length); // Output: 3

Looping Through an Array


for (let i = 0; i < fruits.length; i++) {
  console.log(fruits[i]);
}

Common Array Methods

  • push() – Adds an element at the end
  • pop() – Removes the last element
  • shift() – Removes the first element
  • unshift() – Adds an element at the beginning
  • indexOf() – Returns the index of an element

fruits.push("Pineapple");
console.log(fruits); // ["Apple", "Orange", "Mango", "Pineapple"]

JavaScript Set Logic

JavaScript Set objects can be used to perform common set operations like union, intersection, and difference.

1. Union

Combine all unique elements from two sets.

console.log(union); // Set {1, 2, 3, 4, 5}

2. Intersection

Elements common to both sets.

  [...setA].filter(x => setB.has(x))
);

Elements in one set but not in the other.


const difference = new Set(
  [...setA].filter(x => !setB.has(x))
);
console.log(difference); // Set {1, 2}

  • Useful to keep context in callbacks.

Function Closures

A closure is a function that remembers the environment in which it was created.

Example


function outer() {
  let count = 0;
  return function inner() {
    count++;
    console.log(count);
  }
}

const counter = outer();
counter();  // 1
counter();  // 2

Output:

1
2

Explanation

  • The inner function retains access to count even after outer() finishes.
  • Closures allow private variables and data encapsulation.

JS Classes

Classes in JavaScript are templates for creating objects. They encapsulate data with code to work on that data.

Syntax


class MyClass {
  constructor(value) {
    this.value = value;
  }
  method() {
    console.log(this.value);
  }
}

Example


const obj = new MyClass('Hello');
obj.method();  // Hello

Output: Hello

Explanation

  • Classes use constructor for initialization.
  • Methods are added inside the class body.
  • Objects are created using the new keyword.

Class Intro

Introduces the basics of JavaScript classes.

Example


class Person {
  constructor(name) {
    this.name = name;
  }
  greet() {
    console.log('Hi, ' + this.name);
  }
}

const person = new Person('John');
person.greet();  // Hi, John

Output: Hi, John

Explanation

  • constructor initializes object properties.
  • Methods are part of the prototype.

Class Inheritance

Classes can inherit properties and methods from other classes using extends.

Example


class Animal {
  constructor(name) {
    this.name = name;
  }
  speak() {
    console.log(this.name + ' makes a noise.');
  }
}

class Dog extends Animal {
  speak() {
    console.log(this.name + ' barks.');
  }
}

const d = new Dog('Rex');
d.speak();  // Rex barks.

Output: Rex barks.

Explanation

  • extends creates a subclass inheriting from a parent class.
  • Subclass can override methods.
  • super() is used to call the parent constructor (not shown here, but important in more complex cases).

Class Static

Static methods belong to the class itself, not instances, and are called directly on the class.

Example


class MathUtils {
  static square(x) {
    return x * x;
  }
}

console.log(MathUtils.square(5));  // 25

Output: 25

Explanation

  • Static methods cannot be called on instances.
  • Used for utility functions related to the class.

JS HTML DOM

The HTML Document Object Model (DOM) is a programming interface for HTML documents, representing the page so programs can change structure, style, and content.

Key Points

  • Represents HTML elements as objects.
  • Allows JavaScript to manipulate page content and structure dynamically.

DOM Intro

The DOM represents the document as nodes and objects so that programming languages can interact with the page.

Example


console.log(document.title);  // Logs the page title

Output: The current page's title

Explanation

  • document is the root object representing the webpage.
  • Using DOM, you can read or modify page elements.

DOM Methods

Common methods to access and manipulate DOM elements.

Examples


// Select element by ID
const element = document.getElementById('myElement');

// Select elements by class name
const items = document.getElementsByClassName('item');

// Select elements by CSS selector
const firstItem = document.querySelector('.item');

// Create new element
const newDiv = document.createElement('div');

// Append element
document.body.appendChild(newDiv);

Explanation

  • getElementById, getElementsByClassName, and querySelector retrieve elements.
  • createElement creates new nodes.
  • appendChild adds elements to the DOM.

DOM Document

The document object represents the entire HTML document.

Common Properties


document.title       // Gets or sets the page title
document.URL         // Gets the page URL
document.body        // Accesses the body element
document.forms       // Collection of forms

Example


console.log(document.title);
document.title = 'New Title';

Explanation

  • Manipulating document properties updates the page metadata or content.

DOM Elements

Elements are the building blocks of the DOM, representing HTML tags.

Example


const elem = document.createElement('p');
elem.textContent = 'Hello World';
document.body.appendChild(elem);

Output: Adds a paragraph with text "Hello World" to the page.

Explanation

  • Elements represent nodes in the DOM tree.
  • You can create, modify, and remove elements dynamically.
