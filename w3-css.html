CSS Introduction

CSS is the language we use to style a Web page.

What is CSS?

  • CSS stands for Cascading Style Sheets
  • CSS describes how HTML elements are displayed on screen, paper, or other media
  • CSS saves a lot of work and can control the layout of multiple web pages all at once
  • External stylesheets are stored in .css files

Why Use CSS?

CSS is used to define styles for your web pages, including design, layout, and display variations for different devices and screen sizes.

CSS Solved a Big Problem

HTML was never intended to contain tags for formatting web pages. It was created to describe the content of a page, like:

<h1>This is a heading</h1>
<p>This is a paragraph.</p>

When tags like <font> and color attributes were added in HTML 3.2, it created difficulties for developers. Every page required manual styling, which was hard to maintain.

To solve this issue, the World Wide Web Consortium (W3C) introduced CSS. It moved the styling away from the HTML, making pages cleaner and easier to manage.

Flexbox Intro

Flexbox, or the Flexible Box Layout, is a CSS3 layout model that allows items within a container to be automatically arranged depending on screen size and available space.

‚úÖ Benefits of Flexbox:

  • Easy alignment of items
  • Responsive design-friendly
  • No need to use floats or positioning

üîß Basic Syntax:


.container {
  display: flex;
}

After setting display: flex on a container, its child elements become flex items and can be manipulated using various flex properties.

Flex Container

A Flex Container is the parent element that has display: flex or display: inline-flex set. All its direct children become flex items.

üìå Basic Example:


.container {
  display: flex;
  flex-direction: row;
  justify-content: space-between;
  align-items: center;
}

‚öôÔ∏è Common Flex Container Properties:

  • flex-direction - row | column | row-reverse | column-reverse
  • justify-content - flex-start | center | space-between | space-around
  • align-items - stretch | center | flex-start | flex-end
  • flex-wrap - nowrap | wrap | wrap-reverse
  • align-content - space-between | center | stretch

These properties help arrange and align flex items inside the container efficiently.

Flex Items

Flex Items are the direct children of a flex container. You can control their size, order, and alignment using various properties.

üìå Basic Styling of Flex Items:


.item {
  flex: 1;
  order: 2;
  align-self: center;
}

‚öôÔ∏è Common Flex Item Properties:

  • order - Changes the order of the item (default is 0)
  • flex-grow - Defines how much the item grows relative to the others
  • flex-shrink - Defines how much the item shrinks if necessary
  • flex-basis - Sets the initial size before remaining space is distributed
  • flex - A shorthand for grow, shrink, and basis (e.g., flex: 1 1 auto)
  • align-self - Overrides the container‚Äôs align-items for a single item

Use these properties to control how each flex item behaves inside the container.

Flex Responsive

Flexbox makes it easy to create responsive layouts that adapt to various screen sizes without using floats or positioning.

üì± Example: Responsive Flexbox Layout


.container {
  display: flex;
  flex-wrap: wrap;
}

.item {
  flex: 1 1 200px;
  margin: 10px;
}

üîÅ Key Concepts:

  • flex-wrap: wrap; - Allows items to wrap to the next line if needed
  • flex-basis: - Sets minimum width before wrapping (e.g., 200px)
  • flex: - Shorthand for grow, shrink, and basis (e.g., flex: 1 1 200px)
  • media queries: - Further adjust layout based on screen width

üß™ Example with Media Queries


@media (max-width: 600px) {
  .item {
    flex: 1 1 100%;
  }
}

This ensures your layout adjusts beautifully across desktops, tablets, and phones.

CSS Grid Intro

CSS Grid Layout is a powerful two-dimensional layout system for the web. It lets you design web pages using rows and columns.

üìö Why Use Grid?

  • ‚úÖ Easy to build complex layouts
  • ‚úÖ Control over both rows and columns
  • ‚úÖ Clean and readable code

üß™ Example: Basic Grid Container


.container {
  display: grid;
  grid-template-columns: auto auto auto;
  gap: 10px;
}
.item {
  background-color: lightblue;
  padding: 20px;
  text-align: center;
}

This creates a 3-column grid layout with gaps between grid items.

CSS Grid Columns & Rows

In CSS Grid, you can define the number and size of columns and rows using the grid-template-columns and grid-template-rows properties.

üì¶ Define Columns and Rows


.container {
  display: grid;
  grid-template-columns: 100px 200px auto;
  grid-template-rows: 50px 100px;
}

This creates a grid with 3 columns (fixed and auto width) and 2 rows with specific heights.

üéØ Repeat Function

Use repeat() to simplify repetitive column/row definitions:


.container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
}

This creates 3 equal-width columns using fractional units.

üî¢ Units Used

  • px ‚Äì pixels
  • % ‚Äì percentage
  • fr ‚Äì fractional unit of available space
  • auto ‚Äì automatic sizing

CSS Grid Container

To use CSS Grid Layout, you must first define a container element as a grid with display: grid. All direct children of the grid container become grid items.

üì¶ Defining a Grid Container


.container {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr;
  gap: 10px;
}

This creates a grid with three equal-width columns and a 10px gap between rows and columns.

üîß Additional Properties

  • grid-template-columns ‚Äì defines number and size of columns
  • grid-template-rows ‚Äì defines number and size of rows
  • gap ‚Äì shorthand for row-gap and column-gap
  • justify-content ‚Äì aligns the grid along the row axis
  • align-content ‚Äì aligns the grid along the column axis

‚úÖ Example Layout


.container {
  display: grid;
  grid-template-columns: 100px 1fr 100px;
  grid-template-rows: auto auto;

  padding: 20px;
